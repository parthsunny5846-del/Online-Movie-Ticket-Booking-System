# Online-Movie-Ticket-Booking-System
PROJECT LINK:https://drive.google.com/file/d/1SpHyM_GWH5ElMzBOlIQh9vXD3YiKV-WR/view?usp=drive_link
Online Movie Ticket Booking System (Java) is a desktop application that allows users to browse movies, view showtimes, select seats, and book tickets. Built using core Java, OOP concepts, GUI (Swing/AWT), and JDBC, it provides a simple interface to simulate real-world cinema booking.
